# Auction-Contract
Auction and Biding Contract Tutorial

This smart contract is designed on Auction and Biding concept! Here I will share the complete code to design a Auction Contract which allows to bid on NFT ERC 721 and ERC 20 tokens and also you can send your bid with your ethereum based tokens.

This contract has also used Openzeppelin Library to interact with ERC20 and ERC721 contract. 

Function of the contract:

1. Create Auction 
2. Bid on Auction
3. Claim the rewards 
4. Withdrawn from contract 
5. Change and Update Auction Endtime
